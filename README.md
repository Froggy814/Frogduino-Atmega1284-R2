# Frogduino Atmega1284 R2

The Frogduino Atmega1284 is a Atmega1284 board based off of the Arduino Uno and Mega, I needed more ports but a through hole chip, so I made this... A Frogduino!

The labels are based off of the Mighty Core/Sanguino port layout, so you would want to bootload with one of those loaders if you are not changing the pins.
I have included a BOM (Bill Of Materials) and all of the parts are off mouser.com. If you want me to add anything please let me know either on forum.arduino.cc (I am under Froggy814 there) Or through pull requests.

I will be adding photos soon once I order the board. I will try to keep updating the board as I go along, so... Go and get a Frogduino!!!
